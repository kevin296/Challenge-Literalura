# LiterAlura
---------------------------------------------------------------------------------
LiterAlura es una aplicación de consola para gestionar una colección de libros y autores. Permite buscar libros por título, listar libros y autores registrados, generar estadísticas de descargas, y más.
---------------------------------------------------------------------------------
